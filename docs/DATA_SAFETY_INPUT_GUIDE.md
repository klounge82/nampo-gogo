# Google Play Data Safety 입력 가이드

본 문서는 Google Play Console 에 앱 배포 시 작성해야 하는 데이터 보안 및 개인정보 수집 선언 가이드라인입니다.

---

## 2. 기타 데이터 보호 선언 지침
- **전송 중 암호화**: 예, 남포 GoGo 앱은 모든 사용자 데이터를 안전한 **HTTPS(SSL)** 연결 프로토콜을 경유하여 암호화 전송합니다.
- **데이터 삭제 지원**: 예, 앱 설정 프로필 내 회원탈퇴 기능 및 별도의 웹 고객지원 경로를 통해 명시적인 데이터 삭제 요청을 지원합니다.
- **금융/결제 정보**: 아니오, MVP 단계에서는 실제 PG 결제를 연동하지 않고 모의 결제(Mock PG)를 수행하므로, 사용자의 실제 신용카드 번호나 금융 정보를 수집하거나 전송하지 않습니다. (추후 실제 PG 상용화 시 해당 지표 갱신 필수)

## 3. 개인정보처리방침 및 데이터 삭제 요청 URL 선언
- **개인정보처리방침 URL**: `https://nampo-gogo.netlify.app/privacy/`
  - Play Console '앱 콘텐츠' -> '개인정보처리방침' 항목에 필히 위 URL을 등록해 주어야 합니다.
- **데이터 삭제 요청 URL**: `https://nampo-gogo.netlify.app/delete-account/`
  - Play Console '앱 콘텐츠' -> '데이터 삭제 요청' 항목에 위 URL을 기재하여 이메일 및 앱 외 경로를 통한 삭제 요청 프로세스를 보장합니다.
- **베타용 지원 문의 주소**: `https://nampo-gogo.netlify.app/support/` / `klounge@kakao.com`
  - 문의 대응을 위한 공식 채널로 매핑됩니다. (정식 론칭 전 도메인 메일로 이관 예정)
