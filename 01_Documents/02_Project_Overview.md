# 02. Project Overview

## 프로젝트명

남포 GoGo

## 공식 프로젝트명

남포 GoGo 청사진 v2.0

## 프로젝트 목적

AI를 활용하여 관광객과 남포동 소상공인을 연결하고 지역 상생을 실현하는
플랫폼을 구축한다.

## 핵심 사용자

-   관광객
-   사장님
-   직원
-   관리자

## MVP 핵심 기능

-   AI 추천
-   지도
-   다국어 번역
-   미션
-   쿠폰
-   QR/GPS 인증
-   리뷰
-   관리자 시스템

## 프로젝트 원칙

-   기억보다 기록
-   MVP는 가볍게 유지
-   현실 운영 중심
-   확장 기능은 별도 관리
-   모든 변경은 기록

## 개발 준비 상태

기획 및 설계 완료 개발 문서 작성 진행 중
