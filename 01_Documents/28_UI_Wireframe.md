# 28. UI 와이어프레임

## 목적
개발자와 디자이너가 화면 구조를 빠르게 이해하기 위한 저충실도(Wireframe) 설계입니다.

---

# 1. 관광객 앱

```text
[스플래시]
      │
      ▼
[소셜 로그인]
      │
      ▼
[추가정보 입력]
      │
      ▼
[홈]
 ├─ AI 추천
 ├─ 지도
 ├─ 미션
 ├─ 쿠폰
 └─ 마이페이지
```

### 홈 화면

```text
┌──────────────────────────────┐
│ 🔍 검색                      │
├──────────────────────────────┤
│ AI 추천 카드                 │
├──────────────────────────────┤
│ 인기 매장                    │
├──────────────────────────────┤
│ 진행 중 미션                 │
├──────────────────────────────┤
│ 하단 메뉴                    │
│ 홈│지도│미션│쿠폰│MY        │
└──────────────────────────────┘
```

---

# 2. 사장님 앱

```text
로그인
  │
대시보드
 ├─ 매장관리
 ├─ 미션관리
 ├─ 쿠폰관리
 ├─ 리뷰관리
 └─ 통계
```

---

# 3. 직원 화면

```text
로그인
 │
QR 인증
 │
쿠폰 승인
 │
완료
```

---

# 4. 관리자

```text
대시보드
 ├─ 승인관리
 ├─ 회원관리
 ├─ 매장관리
 ├─ 신고관리
 ├─ 운영정책
 └─ 통계
```

---

## 디자인 원칙

- 큰 버튼 중심
- 한 손 사용 최적화
- 사장님은 2~3번 터치 이내 완료
- 직원은 QR/쿠폰만 빠르게 수행
- 관광객은 지도와 AI 추천 우선
